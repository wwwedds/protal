# 库存仿真前端front

前端特性：不仅支持lui（物流UI组件库）， 支持element ui， 同时支持价值供应链UI组件库vsc-front-ui。

## 项目开始安装依赖
### 注：如果没有安装jnpm执行
```
npm install @jd/jnpm -g --registry=http://registry.m.jd.com

```

### 安装jnpm后执行
```
jnpm install

```

### 运行环境
```
npm run serve
```

### 打包时，执行
```
npm run build
```

### lui前端，请参考：
http://lui.jd.com/#/
